# Try publish package to npm

Create `tsconfig.json`:

[Youtube Ben Award tuto](https://www.youtube.com/watch?v=Ufwbp838yZA)

Npm package name: `td-tsconfig`:

- Install package
  ```bash
  $ yarn add td-tsconfig
  # or with npm
  $ npm i td-tsconfig
  ```

- Command:
  ```bash
  $ yarn td-tsconfig
  # or with npx
  $ npx td-tsconfig
  ```
