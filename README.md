Npm package name: `ts-tsconfig`:

- Install package
  ```bash
  $ yarn add ts-tsconfig
  # or with npm
  $ npm i ts-tsconfig
  ```

- Command:
  ```bash
  $ yarn td-tsconfig

  $ npx ts-tsconfig
  ```
